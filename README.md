# PictureReSize
フォルダ単位で簡単に画像の解像度を拡大縮小出来るソフトです。  

# ライセンス
・変換処理には[SixLabors.ImageSharp](https://github.com/SixLabors/ImageSharp)を使用しています。ImageSharpは[SixLaborsSplitLicense1.0](https://github.com/SixLabors/ImageSharp/blob/main/LICENSE)で配布されています。  
・アプリケーション更新処理には[VeloPack](https://github.com/velopack/velopack/)を使用しています。VeloPackは[MIT License](https://github.com/velopack/velopack/blob/develop/LICENSE)で配布されています。
